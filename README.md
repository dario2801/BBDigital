# Sobre la Web
1.  Este Proyecto es un Ejercicio Tècnico de BBDigital, consiste en el desarrollo de una pègina de ventas de artìculos para distintas categorìas las 
cuales son: **Para niños, Para niñas, Para la casa y Para juguetes**.Al inicio de la web se muestra las opciones de categorias, si alguna es seleccionada
se desplegará todos los artículos disponibles, en caso contrario se irán mostrando (9) productos aleatoriamente con respecto a las categorías existentes.

2. El objetivo de la página es poder mostrar disponibilidad de productos para su venta, están catalogaados por rango y categoría.

3. Aún se encuentra en proceso de desarrollo, para agregar funcionalidades que brinden una mejor experiencia de usuario.Para su visualización no se necesita
intalación de extensiones externas.
